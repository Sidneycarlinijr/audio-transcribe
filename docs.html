Scripts

transcribe_audio.py Principal

Script completo: divide audios longos, transcreve cada segmento via Gemini, junta tudo e salva como Markdown com tema automatico.

python transcribe_audio.py audio.mp3 python transcribe_audio.py --folder ~/Audios python transcribe_audio.py audio.mp3 --segment-duration 5 python transcribe_audio.py audio.mp3 --output ~/Transcricoes

Saida: ~/Documentos/Transcricoes/2026-04-02 — Tema.md

split_audio.py Utilitario

Apenas divide audios longos em segmentos menores (sem transcricao). Util para pre-processar antes de transcrever ou para enviar partes menores manualmente.

python split_audio.py audio.mp3 python split_audio.py --folder ~/Audios python split_audio.py audio.mp3 --segment-duration 15 python split_audio.py audio.mp3 --output ~/Partes

Saida: audio_partes/audio_parte_01.mp3, audio_parte_02.mp3, ...

transcribe_segment.py Leve

Transcreve um unico arquivo de audio. Inclui retry com backoff exponencial para rate limiting. Saida via stdout — ideal para pipelines e composicao com outros scripts.

python transcribe_segment.py audio.mp3 > transcricao.txt

Retry: 3 tentativas, backoff de 40s * tentativa

Setup

Criar venv e instalar dependencias

python3 -m venv ~/venvs/transcribe source ~/venvs/transcribe/bin/activate pip install google-generativeai pydub

Instalar ffmpeg

sudo apt install ffmpeg

Configurar API Key

Veja a secao detalhada abaixo.

# Adicione ao ~/.zshrc ou ~/.bashrc export GEMINI_API_KEY="sua_key_aqui"

Adicionar alias (opcional)

# Evita ativar venv toda vez alias transcribe="~/venvs/transcribe/bin/python transcribe_audio.py"

Configurar API Key

Como gerar a chave

Acesse o Google AI Studio

Abra aistudio.google.com/app/apikey e faca login com sua conta Google.

Aceite os termos

Na primeira visita, aceite os Termos de Servico de IA Generativa e confirme sua regiao.

Clique em "Create API Key"

Um projeto Google Cloud e criado automaticamente (ou selecione um existente). Nao precisa configurar nada no GCP manualmente.

Copie a chave

A key comeca com AIza seguido de caracteres alfanumericos. Copie imediatamente — ela nao sera exibida novamente.

Configure no terminal

# Adicione ao ~/.zshrc (permanente) export GEMINI_API_KEY="AIzaSy..." # Recarregue o shell source ~/.zshrc

Free Tier — Limites

O tier gratuito funciona sem cadastrar cartao de credito.

Limite Valor
Requests por minuto 10 RPM
Requests por dia 250 RPD
Tokens por minuto 250.000 TPM
Janela de contexto 1.000.000 tokens
Custo $0
Privacidade no free tier

No tier gratuito, seus prompts e respostas podem ser usados pelo Google para melhorar os modelos. Se voce transcreve audios com conteudo sensivel, considere usar o tier pago onde seus dados nao sao usados para treinamento.

Na pratica

250 requests/dia e suficiente para transcrever varios audios por dia. Um audio de 1 hora dividido em segmentos de 10 minutos usa 6 requests. O rate limit de 10 RPM pode causar espera entre segmentos — o transcribe_segment.py ja lida com isso automaticamente via retry.

Tier Pago — Custos

Para uso intensivo ou com dados sensiveis. Ativacao: minimo de $10 pre-pagos no AI Studio.

Tipo Preco (por 1M tokens)
Input — texto/imagem $0.30
Input — audio $1.00
Output (transcricao) $2.50
Input audio (Batch API, ~50% off) $0.50
Estimativa de custo

Audio de 1 hora transcreve por centavos. O custo principal e o input de audio ($1/1M tokens). Na pratica, mesmo transcrevendo dezenas de reunioes por mes, o custo fica abaixo de $5.

Troubleshooting

Erro: GEMINI_API_KEY nao definida

Configure a variavel de ambiente:

export GEMINI_API_KEY="sua_key"

Erro: pydub nao instalado

Ative o venv e instale:

source ~/venvs/transcribe/bin/activate pip install pydub google-generativeai

Erro: ffmpeg nao encontrado

sudo apt install ffmpeg

Rate limit / 429 Error

transcribe_segment.py ja tem retry automatico. Para transcribe_audio.py, reduza segmentos:

python transcribe_audio.py audio.mp3 --segment-duration 5
