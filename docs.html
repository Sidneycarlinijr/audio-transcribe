Como Usar

Com Claude Code Recomendado

Clone o repo e abra com Claude Code. A skill de transcricao e descoberta automaticamente.

Scripts

transcribe_audio.py Principal

Script completo: envia audio para Deepgram, recebe transcricao com diarizacao nativa, extrai tema e salva como Markdown.

python transcribe_audio.py audio.mp3 python transcribe_audio.py --folder ~/Audios python transcribe_audio.py audio.mp3 --output ~/Transcricoes

Saida: ~/Documentos/Transcricoes/2026-04-02 — Tema.md

split_audio.py Utilitario

Apenas divide audios longos em segmentos menores (sem transcricao). Util para pre-processar ou enviar partes menores manualmente.

python split_audio.py audio.mp3 python split_audio.py --folder ~/Audios python split_audio.py audio.mp3 --segment-duration 15 python split_audio.py audio.mp3 --output ~/Partes

Saida: audio_partes/audio_parte_01.mp3, audio_parte_02.mp3, ...

transcribe_segment.py Leve

Transcreve um unico arquivo de audio. Saida via stdout — ideal para pipelines e composicao com outros scripts.

python transcribe_segment.py audio.mp3 > transcricao.txt

Setup

Criar venv e instalar dependencias

python3 -m venv ~/venvs/transcribe source ~/venvs/transcribe/bin/activate pip install deepgram-sdk pydub

Instalar ffmpeg

sudo apt install ffmpeg

Configurar API Key

Veja a secao detalhada abaixo.

# Adicione ao ~/.zshrc ou ~/.bashrc export DEEPGRAM_API_KEY="sua_key_aqui"

Adicionar alias (opcional)

# Evita ativar venv toda vez alias transcribe="~/venvs/transcribe/bin/python transcribe_audio.py"

Configurar API Key

Como gerar a chave

Acesse o Deepgram Console

Abra console.deepgram.com e crie uma conta.

Crie um projeto

Um projeto padrao e criado automaticamente no signup.

Gere uma API Key

Va em Settings > API Keys e clique em "Create a New API Key". Selecione as permissoes de "Member".

Copie a chave

Copie imediatamente — ela nao sera exibida novamente.

Configure no terminal

# Adicione ao ~/.zshrc (permanente) export DEEPGRAM_API_KEY="sua_key_aqui" # Recarregue o shell source ~/.zshrc

Troubleshooting

Erro: DEEPGRAM_API_KEY nao definida

Configure a variavel de ambiente:

export DEEPGRAM_API_KEY="sua_key"

Erro: deepgram-sdk nao instalado

Ative o venv e instale:

source ~/venvs/transcribe/bin/activate pip install deepgram-sdk pydub

Erro: ffmpeg nao encontrado

sudo apt install ffmpeg
